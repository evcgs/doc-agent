# 文档处理智能体 - 专业的文档全生命周期处理Agent V1.0
## 🎯 项目说明
DocAgent是专门负责文档管理的智能助手，专注于文档读写、Wiki编辑、知识库管理和多维表格操作，支持飞书文档、Wiki、多维表格、云存储等多源文档的全生命周期管理。可以作为独立的OpenClaw Agent使用，也可以通过Docker部署。

### 解决的核心痛点
- ❌ 多平台文档管理麻烦，需要在不同系统之间切换
- ❌ 知识库内容零散，难以统一管理和检索
- ❌ 多维表格操作复杂，需要编写复杂的公式和脚本
- ❌ 云存储文件杂乱，查找和整理耗时耗力
- ❌ 文档协作效率低，需要反复同步和确认

## ✨ 核心特性
### 📝 文档管理
- ✅ 读取和查看飞书文档、Word、PDF、Markdown等多种格式
- ✅ 创建新的飞书文档，支持模板快速生成
- ✅ 修改和更新现有文档，支持批量替换内容
- ✅ 删除文档（需用户确认，防止误删）
- ✅ 文档格式转换：支持Markdown↔Word↔PDF互转

### 🧠 知识库操作
- ✅ Wiki页面导航和智能搜索，快速定位内容
- ✅ Wiki页面创建和编辑，支持模板导入
- ✅ 知识库目录管理，自动整理目录结构
- ✅ 知识内容整理和归档，自动生成知识图谱
- ✅ 知识库内容查重和去重，避免冗余

### 📊 多维表格操作
- ✅ Bitable数据读取和复杂查询，支持多条件筛选
- ✅ Bitable记录创建和批量更新，提高数据录入效率
- ✅ Bitable表格创建和配置，自动生成常用字段
- ✅ 数据统计和报表生成，支持可视化图表
- ✅ 多维表格数据导出，支持Excel、CSV等格式

### ☁️ 云存储管理
- ✅ 飞书云盘文件管理，支持上传、下载、删除
- ✅ 文件夹创建和自动整理，按规则分类文件
- ✅ 文件权限管理，批量设置访问权限
- ✅ 文件搜索和筛选，支持按名称、类型、时间检索
- ✅ 大文件分片上传和断点续传，提高传输效率

## 🚀 安装方式
### 方式1：OpenClaw技能市场安装（推荐）
1. 打开OpenClaw控制面板 → 技能市场
2. 搜索「文档处理智能体」
3. 点击「安装」即可自动完成部署

### 方式2：本地安装
```bash
# 1. 下载到Agent目录
cd ~/.openclaw/agents
git clone https://github.com/evcgs/doc-agent.git

# 2. 安装依赖
cd doc-agent
npm install
```

### 方式3：Docker一键部署
```bash
docker run -d \
  --name doc-agent \
  -v ~/.openclaw/doc-agent-config:/app/config \
  evcgs/doc-agent:v1.0.0
```

## 📝 使用指南
### 基础用法
直接用自然语言描述需求即可：
```
"帮我读取这个飞书文档的内容，提取核心要点"
"在知识库创建一个新的产品需求文档页面，使用产品需求模板"
"给这个多维表格添加10条测试数据，字段包括姓名、年龄、部门"
"整理云盘里的项目文档，按项目分类放到不同的文件夹"
"把这份Markdown文档转换成PDF格式，保留原有格式"
```

### 高级用法
可以指定特殊要求：
```
"查询多维表格中2026年3月的销售数据，生成统计报表，包含折线图和饼图"
"搜索知识库中所有关于医疗大模型的内容，整理成一份综述文档"
"批量修改飞书文档中所有的"旧品牌名"为"新品牌名"，共50份文档"
"设置云盘里的项目文档文件夹权限，只允许项目组成员编辑"
```

## 📋 典型使用场景
### 1. 知识库管理
> 统一管理企业知识库，自动整理和归档
```
"整理知识库中所有的产品文档，按产品线分类，生成目录索引"
```

### 2. 数据报表生成
> 自动从多维表格提取数据，生成可视化报表
```
"统计本季度的销售数据，生成季度销售报表，包含趋势分析和同比环比"
```

### 3. 批量文档处理
> 批量处理大量文档，提高工作效率
```
"把所有旧版的产品文档都加上「已归档」水印，移动到归档文件夹"
```

### 4. 云存储整理
> 自动整理杂乱的云存储文件，提升查找效率
```
"整理云盘里的所有图片，按拍摄时间分类到不同的文件夹"
```
